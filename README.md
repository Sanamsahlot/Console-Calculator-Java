# Smart Console Calculator (Java) 🚀

An advanced, menu-driven console calculator application built with Core Java. This project demonstrates strong foundations in programming logic, structured control flow, and exception handling principles.

## 🛠️ Key Features
- **Comprehensive Arithmetic Suite:** Supports Addition, Subtraction, Multiplication, Division, Modulus, Power ($x^y$), Square Root ($\sqrt{x}$), and Percentage calculations.
- **Fail-Safe Robustness:** Preventative input validation logic for critical runtime exceptions like division by zero and negative square root calculations.
- **Dynamic UX:** Built using an automated persistent loop with explicit exit conditions.

## 💻 Tech Stack & Concepts Demonstrated
- **Language:** Java (JDK 26 Architecture)
- **Input Stream Handling:** `java.util.Scanner`
- **Control Layout:** Structured `while(true)` loops combined with explicit multi-branch `switch-case` statements.
- **Version Control:** Managed via Git pipeline architecture and hosted publicly on GitHub.
